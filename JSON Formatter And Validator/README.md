# JSON Formatter & Validator

A simple web app that helps developers quickly format, validate, minify, and copy JSON data.

Built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## Features

* Format JSON with proper indentation
* Validate JSON input
* Minify JSON
* Copy output to clipboard
* Clear input and output fields
* Load sample JSON for testing
* Responsive dark-themed design

## How to Use

1. Paste your JSON into the input area.
2. Click **Format** to beautify the JSON.
3. Click **Minify** to remove unnecessary whitespace.
4. Use **Copy Output** to copy the result.
5. Click **Sample JSON** to load example data.
6. Click **Clear** to reset everything.
